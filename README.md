# My-i3-config
Just a backup of my i3-config file solving my needs.

for enabling tapping the touchpad `xinput` : 
```bash
xinput list-props [device-number]
xinput set-prop <device> [--type=atom|float|int] [--format=8|16|32] <property> <val> [<val> ...]
```
